# /market-analysis - Local Market Intelligence

Command for analyzing local market prices and trends in your territory.

## Usage

```bash
/market-analysis "organic produce"
/market-analysis "tech services pricing"
/market-analysis "local restaurant market"
```

## Agent Spawned

This command activates the **Market Analyst** agent with capabilities:

- Analyze local market prices
- Compare weekly/monthly trends
- Predict short-term fluctuations

3. **Track trends** rather than instant prices
