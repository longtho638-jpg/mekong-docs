# /content-marketing - Content Strategy Assistant

Build a complete **Content Marketing** strategy for your business.

## Usage

```bash
/content-marketing "e-commerce blog"
/content-marketing "B2B thought leadership"
/content-marketing "local business social media"
```

## Output Includes

- Content pillars and themes
- Channel strategy
- Content calendar template
- SEO keyword recommendations
- Performance metrics

## Best Practices

1. **Quality over quantity** - Focus on value
2. **Repurpose content** - One idea, many formats
3. **Track engagement** - Optimize based on data
