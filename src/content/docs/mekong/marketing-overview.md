# Marketing Tools

Your complete marketing toolkit within Agency OS.

## Available Commands

| Command | Description |
|---------|-------------|
| `/marketing-plan` | Create comprehensive marketing strategy |
| `/content-marketing` | Build content strategy |
| `/social-media` | Generate social post ideas |
| `/messaging` | Craft value propositions |
| `/pr-strategy` | Plan PR campaigns |

## Getting Started

1. Start with `/market-research` to understand your landscape
2. Use `/customer-profile` to define your audience
3. Build your `/marketing-plan`
4. Execute with the other tools

## Best Practices

- **Work systematically** - Strategy first, tactics second
- **Test and learn** - Marketing is iterative
- **Stay consistent** - Brand building takes time
