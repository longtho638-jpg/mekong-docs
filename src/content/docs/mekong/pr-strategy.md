# /pr-strategy - Public Relations Planner

Build an effective **PR Strategy** for your brand.

## Usage

```bash
/pr-strategy "product launch"
/pr-strategy "crisis management"
/pr-strategy "thought leadership"
```

## Output Includes

- PR objectives
- Target media outlets
- Story angles
- Press release templates
- Influencer outreach plan

## Best Practices

1. **Build relationships** before you need them
2. **Be newsworthy** - Create genuine stories
3. **Measure impact** - Track media mentions
