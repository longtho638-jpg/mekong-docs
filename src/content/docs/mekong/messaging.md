# /messaging - Value Proposition Builder

Create compelling **Marketing Messages** and value propositions.

## Usage

```bash
/messaging "SaaS productivity tool"
/messaging "eco-friendly products"
/messaging "professional services"
```

## Output Includes

- Unique value proposition
- Taglines and slogans
- Benefit statements
- Call-to-action variations
- Elevator pitch

## Best Practices

1. **Lead with benefits** - What's in it for them?
2. **Be specific** - Vague claims don't convert
3. **Test variations** - A/B test your messaging
