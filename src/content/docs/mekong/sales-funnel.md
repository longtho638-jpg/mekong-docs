# /sales-funnel - Funnel Builder

Create high-converting **Sales Funnels** for your business.

## Usage

```bash
/sales-funnel "SaaS free trial to paid"
/sales-funnel "e-commerce checkout"
/sales-funnel "lead magnet to consultation"
```

## Output Includes

- Funnel stages mapping
- Conversion optimization tips
- Email sequence framework
- A/B testing suggestions
- Metric benchmarks

## Best Practices

1. **Map the journey** - Understand each step
2. **Reduce friction** - Remove unnecessary steps
3. **Track everything** - Measure conversion at each stage
