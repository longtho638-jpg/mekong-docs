# /social-media - Social Media Content Generator

Generate creative **Social Media** content ideas for any platform.

## Usage

```bash
/social-media "Instagram for fitness brand"
/social-media "LinkedIn for B2B services"
/social-media "TikTok for restaurant"
```

## Output Includes

- Post concepts and hooks
- Visual content ideas
- Hashtag recommendations
- Posting schedule suggestions
- Engagement tactics

## Best Practices

1. **Know your platform** - Each has different rules
2. **Be authentic** - Don't just sell, connect
3. **Engage daily** - Social is a two-way street
